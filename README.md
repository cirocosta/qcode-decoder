# qcode js!

> Decodes QRCode in the browser / node

## Quickuse

Download it as a dependency

```sh
$ bower install qcode
```

attach it to your html file

```html
<script src="../bower_components/qcode-js/build/qcode.min.js"></script>
```

and use it!

